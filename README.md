# live.itrip  project

项目结构：
1. admin 服务管理后台
2. common 项目中的公共类库
3. redis 实现缓存，支持分布式
4. sso 登录及token验证
5. wechat 微信 sdk 封装
6. web  itrip.live websit

使用技术：
1. spring + springmvc + mybatis + maven
2. html5（bootstrap） + jquery
3. Redis（缓存服务器）
4. Shiro（权限框架）
5. Dubbo（调用系统服务） + Zookeeper（分布式应用程序协调服务）
6. Solr（搜索引擎）
7. FastDFS（文件服务器



参考项目：
1. H+ 后台主题UI框架
2. goshop2 https://github.com/pzhgugu/goshop2
